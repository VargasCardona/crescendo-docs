Descripción del Proyecto

El Utilitario Traspaso Crescendo es un sistema desarrollado para automatizar el proceso de traspaso de cotizantes desde fondos privados hacia el fondo público Crescendo en Colombia. Su objetivo principal es agilizar este proceso, reducir tiempos de semanas a horas/días, y garantizar el cumplimiento de normativas financieras y de salud.

El sistema procesa archivos CSV con datos de cotizantes y embargos, valida requisitos, gestiona colas de prioridad, genera una lista negra de cotizantes embargables, y se integra con entidades gubernamentales mediante protocolos seguros.

Beneficios Clave:

  • Reducción del tiempo de traspaso
  • Liberación de recursos humanos
  • Cumplimiento normativo automatizado

Características Técnicas:

  • Procesamiento de archivos CSV
  • Validación automática de requisitos
  • Gestión de colas de prioridad
  • Integración con entidades gubernamentales

Requisitos Funcionales

Nuestro sistema incorpora tecnología de vanguardia para garantizar un proceso eficiente y seguro

RF-001

Lectura de Archivos CSV

El sistema debe leer archivos CSV (cotizantes.csv y embargos.csv) proporcionados por Crescendo y entidades gubernamentales.

Detalles:
  • Entradas: Archivos CSV con formatos predefinidos
  • Salidas: Estructuras Map en memoria
  • Validación: Verificar existencia y formato
RF-002

Validación Automática de Traspasos

Aplicar reglas de negocio para aprobar o rechazar traspasos.

Detalles:
  • Reglas: Edad mínima (18 años), historial sin deudas
  • Salidas: Lista de válidos/inválidos con motivos
  • Ejemplo: "Edad insuficiente"
RF-003

Gestión de Cola de Prioridad

Ordenar traspasos aprobados en PriorityQueue basada en criterios.

Detalles:
  • Prioridad: Mayor edad y tiempo en fondo privado
  • Implementación: PriorityQueue de Java
  • Salida: Cola ordenada para procesamiento
RF-004

Generación de Lista Negra

Crear lista enlazada personalizada para cotizantes con embargos.

Detalles:
  • Entradas: Datos de embargos.csv
  • Salidas: Archivo lista_negra.csv
  • Funcionalidad: Iterador específico para búsqueda
RF-005

Exportación de Resultados

Generar archivos CSV con traspasos aprobados y lista negra.

Detalles:
  • Formatos: traspasos_aprobados.csv, lista_negra.csv
  • Estructura: IDs, nombres, prioridades, detalles de embargos
  • Ubicación: Directorio designado con permisos
RF-006

Integración con Entidades Gubernamentales

Sincronización diaria con archivos de embargos mediante SFTP.

Detalles:
  • Protocolo: SFTP seguro
  • Cifrado: AES-256 para transferencia
  • Automatización: Sin intervención manual
RF-007

Registro de Logs (Auditoría)

Registrar todas las transacciones con detalles completos.

Detalles:
  • Datos: Timestamp, ID, usuario, resultado
  • Ejemplo: "2024-05-06 10:00:00 | ID123 | Validación | Éxito"
  • Almacenamiento: Archivos diarios con respaldo
RF-008

Manejo de Errores

Identificar y gestionar errores en el procesamiento.

Detalles:
  • Tipos: Archivos corruptos, formatos inválidos
  • Acciones: Notificación a administrador
  • Registro: Detalles técnicos en logs
RF-009

Cifrado de Datos Sensibles

Aplicar cifrado AES-256 a archivos con información sensible.

Detalles:
  • Alcance: Archivos de entrada/salida
  • Implementación: Biblioteca estándar Java
  • Claves: Gestión segura de claves
RF-010

Respaldo Automático

Realizar copias diarias de archivos procesados.

Detalles:
  • Frecuencia: Cada 24 horas
  • Retención: 30 días
  • Ubicación: Directorio seguro designado
